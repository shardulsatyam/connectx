# ConnectX 🚀

ConnectX is a modern video meeting and collaboration platform inspired by Google Meet and Zoom.

## ✨ Features

- User Authentication (JWT)
- Secure Login & Registration
- Video Meetings
- Dashboard
- Responsive UI
- MongoDB Database
- Express Backend
- React Frontend

## 🛠️ Tech Stack

### Frontend
- React
- TypeScript
- Vite
- Tailwind CSS

### Backend
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- JWT Authentication

## 📂 Project Structure

```
connectx/
├── frontend/
└── backend/
```

## 🚀 Getting Started

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
